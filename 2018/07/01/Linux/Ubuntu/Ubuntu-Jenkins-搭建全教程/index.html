二 配置jenkins

  1. 获取登录密码
    浏览器打开 http://192.168.x.x:8080

  2. 选择 安装推荐的插件，耐心等待

  3. 修改admin密码.

    若界面是英文，把jenkins服务重启下就好了

  4. 增加root权限
    将root添加到jenkins用户组中，让root用户拥有管理jenkins的权限

    让jenkins用root身份运行，配置文件在/etc/default/jenkins

    jenkins的工作目录在/var/lib/jenkins,把jenkins的默认运行用户改成了root
