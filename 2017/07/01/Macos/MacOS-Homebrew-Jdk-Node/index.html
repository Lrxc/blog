      四 Node Js

  • 默认安装最高的版本
  • 安装指定版本
  1. 如果之前使用过 brew install node 安装过 node，需要使用 brew unlink node 来解绑
  2. 打开命令行，输入 brew search node ，来查询可以安装的 node 版本
  3. 可以看到命令行中出现了包括 node@8、node@10 之类的 node 版本信息。选择自己需要安装的版本，进行安装，如使用命令brew install node@10;
  4. 输入 node -v进行查看
