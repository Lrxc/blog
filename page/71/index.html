Ubuntu ElasticSearch 服务搭建

发表于 2018-07-01 更新于 2021-05-29 分类于 Linux ， Linux
## 环境

  • linux: ubuntu 18.4

  • elasticsearch: 6.5.3

  • jdk: open jdk 1.8

一 服务搭建(压缩包)

  1. 下载elastic服务端

    官网下载地址：https://www.elastic.co/cn/downloads/elasticsearch

  2. 配置

    需要新建一个非root账号，elastic不支持root账号启动

    外网访问：去掉network.host的注释，将它的值改成0.0.0.0，然后重新启动 Elastic, 关闭防火墙！！！

  3. 启动服务

  4. 测试成功

  5. ik分词器

    下载后解压到es的plusins路径下，重启es即可

  5. 修改es配置文件：vim ./config/elasticsearch.yml

  6. 退出容器

  7. 重启容器
