Ubuntu-18-0-4-SonarQube-6-7-x-安装教程

发表于 2018-07-01 更新于 2021-02-02 分类于 Linux ， Linux

##安装环境及版本：

  • 系统：Ubuntu 18.04 LTS
  • JDK: OpenJDK 1.8.0
  • MySql : 5.7
  • SonarQube: SonarQube 6.7.7

##一 下载sonar

  1. 官网地址：https://www.sonarqube.org/downloads/
  • 拉到最下面，选择Show all versions，可选择历史版本

##三 数据库配置

  • 配置sonar的数据库信息(vim conf/sonar.properties) ##四 启动
